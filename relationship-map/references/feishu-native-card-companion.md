# 飞书原生卡片配套层

## 目标

用户没有安装飞书流式卡片插件时，人脉地图不能退化成长 Markdown 文档。必须用飞书原生 interactive card 展示。

## 卡片类型

- 空状态卡片
- 列表卡片
- 详情卡片
- 写入确认卡片
- 示例卡片
- 二级页面卡片

## 二级页面

复杂能力不能平铺。类似“暗影女王”二级页面、投资人分析、政府/协会关系、场景剧本等，都采用一级入口 + 二级页面 + 返回。

## 渲染优先级

1. 流式卡片插件
2. 飞书原生 interactive card
3. 极简纯文本

## v2.1 执行边界

该 companion 不是文档建议，而是必须安装/启用的 Hermes 插件：`plugins/relationship-map-feishu-card`。

验收必须看真实飞书消息类型：`interactive`。如果只是灰底文本、`[查看详情]` 字样、`新增联系人｜搜索` 字样，判定为失败。

该 companion 只是 Hermes 内部渲染层，不单独部署服务器。
